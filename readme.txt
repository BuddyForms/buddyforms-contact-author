Add a button to contact the author to your Posts, from the list of Post or a Post single page.

== Description ==

This add-on integrates with BuddyForms Profesional and gives you the ability to let your user connect with the author of any post directly with no intervention from you.

##Requirement
This extension needs BuddyForms Professional. If you need more than one extension we recommend to take a look to the [ThemeKraft Bundle](https://themekraft.com/products/themekraft-bundle/)

##Features
- Contact the author of a Post
- Let logged in/out user contact the author of a Post based on a configuration.
- Define reusable subject and message.
- Build your message using values from the submitted entry.
- Add the contact action before/after the Post content with a simple click.
- Add a shortcode at any point of a Post to insert the contact action.
- Enable the contact action for specific Form.

##Use-cases
- Let your user contact with you for a good or services you are providing. 
- Let users give you feedback for the post they are reading.

##Source
[Our code](https://github.com/BuddyForms/buddyforms-contact-author) is public on GitHub.
